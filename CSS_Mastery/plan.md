# CSS 工业级精通之路 (Zero to Hero)

> **导师寄语**：既然是小白，我们就把 CSS 拆碎了讲。不跳步，不默认你懂任何术语。我们从“CSS 是什么”开始，一步步把网页变漂亮。

## 🔴 第零阶段：破冰 - CSS 到底是什么 (The Big Picture)
> **目标**：理解 CSS 的作用、基本语法，以及它是如何和 HTML 配合的。

- [x] **00. 初识 CSS**：什么是 CSS？它的语法长什么样？如何把它引入到 HTML 里？
- [x] **01. 选择器入门 (Selectors)**：标签、类、ID 以及 **进阶组合选择器 (>, +, ~)**。

## 🟢 第一阶段：筑基 - 盒子与文本 (Foundations)
> **目标**：建立“万物皆盒子”的直觉，能够独立完成基础图文排版。

- [x] **02. 盒子模型 (Box Model)**：`padding`, `border`, `margin` 的直观理解；`box-sizing` 为什么重要。
- [x] **03. 显示模式 (Display)**：为什么有的元素占一行，有的不占？`block` vs `inline`。
- [x] **04. 颜色与文本**：如何改颜色、改字体大小、改行间距。
- [x] **05. 长度单位**：`px` (死的) vs `rem/vw` (活的)。
- [x] **06. 调试利器 (DevTools)**：如何用浏览器检查样式，实时修改。

## 🟡 第二阶段：进阶 - 现代布局与定位 (Modern Layouts)
> **目标**：掌握 Flex 和 Grid，能够实现 95% 以上的 UI 稿布局需求。

- [ ] **07. 核心布局 A：Flexbox (弹性盒子)**：最常用的排版方式，让元素横着排、竖着排。
- [ ] **08. 定位 (Positioning)**：怎么让元素“飘”起来或者“固定”在某个位置。
- [ ] **09. 核心布局 B：Grid (网格布局)**：像 Excel 一样精准控制网页网格。
- [ ] **10. 权重与层叠 (Specificity)**：当样式冲突时，浏览器听谁的？

## 🔵 第三阶段：质感 - 交互、动画与响应式 (Interactions & Responsiveness)
> **目标**：让页面适配手机，并增加动效。

- [ ] **11. 伪类与伪元素**：鼠标悬停、点击效果，以及 `::before` 等神奇的小技巧。
- [ ] **12. 响应式设计 (RWD)**：如何让网页在手机和电脑上显示不同布局。
- [ ] **13. 变换与动画**：旋转、缩放，以及让元素动起来。

## 🟣 第四阶段：工程化 - 现代 CSS 特性 (Modern Engineering)
> **目标**：掌握 2024-2025 最流行的技术。

- [ ] **14. CSS 变量**：一键换肤的秘密。
- [ ] **15. 现代数学函数**：`calc()`, `clamp()` 让网页自动适应大小。
- [ ] **16. CSS 架构思路**：如何写出好维护、不乱套的 CSS 代码。
