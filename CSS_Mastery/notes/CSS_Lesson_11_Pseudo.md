# Lesson 11: 伪类与伪元素 (Pseudo-classes & Pseudo-elements) - 交互与样式增强

## 1. 伪类（Pseudo-classes）概述

伪类是一种选择器，用于选择元素的**特定状态**或**特定位置**。它们以冒号 `:` 开头，紧跟在选择器后面。

### 1.1 常用伪类分类

| 类别 | 伪类 | 描述 |
|------|------|------|
| **链接状态** | `:link` | 未访问的链接 |
| | `:visited` | 已访问的链接 |
| | `:hover` | 鼠标悬停时 |
| | `:active` | 元素被激活时（点击时） |
| **结构位置** | `:first-child` | 第一个子元素 |
| | `:last-child` | 最后一个子元素 |
| | `:nth-child(n)` | 第n个子元素 |
| | `:nth-of-type(n)` | 同类型元素中的第n个 |
| **表单状态** | `:focus` | 元素获得焦点时 |
| | `:required` | 必填字段 |
| | `:valid` | 有效输入 |
| | `:invalid` | 无效输入 |
| **用户交互** | `:hover` | 鼠标悬停 |
| | `:active` | 元素被激活 |
| | `:focus` | 元素获得焦点 |

## 2. 伪元素（Pseudo-elements）概述

伪元素用于创建**虚拟元素**，可以在文档中插入内容或样式，而不需要修改HTML结构。它们以双冒号 `::` 开头。

### 2.1 常用伪元素

| 伪元素 | 描述 |
|--------|------|
| `::before` | 在元素内容前插入内容 |
| `::after` | 在元素内容后插入内容 |
| `::first-line` | 选择元素的第一行文本 |
| `::first-letter` | 选择元素的第一个字符 |
| `::selection` | 选择用户选中的文本 |

## 3. 伪类与伪元素的区别

| 特性 | 伪类 | 伪元素 |
|------|------|--------|
| 语法 | 单冒号 `:` | 双冒号 `::`（CSS3标准） |
| 作用 | 选择元素的特定状态 | 创建虚拟元素 |
| 数量 | 一个选择器可以有多个伪类 | 一个选择器只能有一个伪元素 |
| 内容 | 不创建新内容 | 可以通过 `content` 属性添加内容 |

## 4. 链接状态伪类（Link Pseudo-classes）

### 4.1 链接状态顺序

**重要**：链接伪类的顺序必须是 `:link` → `:visited` → `:hover` → `:active`，否则可能会出现样式覆盖问题。

```css
/* 正确顺序 */
a:link { color: blue; }
a:visited { color: purple; }
a:hover { color: red; }
a:active { color: green; }
```

### 4.2 实用技巧

- 为链接添加过渡效果，提升用户体验
- 为不同状态设置不同的背景色和文本色
- 可以使用 `text-decoration` 属性控制下划线样式

## 5. 结构伪类（Structural Pseudo-classes）

### 5.1 `:nth-child()` 详解

- `:nth-child(1)` - 第一个子元素
- `:nth-child(even)` - 偶数位置的子元素
- `:nth-child(odd)` - 奇数位置的子元素
- `:nth-child(3n)` - 每隔2个元素（第3、6、9...个）
- `:nth-child(3n+1)` - 从第1个开始，每隔2个元素

### 5.2 实用案例

- 表格隔行变色
- 导航菜单的第一个和最后一个元素特殊样式
- 列表项的奇偶行不同样式

## 6. 伪元素的高级应用

### 6.1 内容生成

使用 `content` 属性可以在伪元素中添加内容：

```css
/* 添加引号 */
.quote::before {
  content: "\"";
}

.quote::after {
  content: "\"";
}

/* 添加图标 */
.icon::before {
  content: "▶";
  margin-right: 8px;
}
```

### 6.2 装饰性元素

- 为卡片添加装饰性边框
- 创建自定义列表项标记
- 实现工具提示效果
- 添加背景图案或渐变

## 7. 表单伪类应用

### 7.1 增强表单用户体验

- 为聚焦的输入框添加特殊样式
- 为必填字段添加视觉提示
- 为有效/无效输入提供即时反馈
- 为提交按钮添加悬停和激活状态

### 7.2 实用技巧

- 使用 `:focus` 伪类提高表单可访问性
- 结合 `:valid` 和 `:invalid` 提供实时验证反馈
- 为不同类型的输入框设置不同的样式

## 8. 伪类与伪元素的组合使用

可以在一个选择器中组合使用多个伪类和一个伪元素：

```css
/* 悬停时显示特殊效果 */
.button:hover::after {
  content: " →";
  transition: all 0.3s ease;
}

/* 聚焦时的输入框提示 */
.input:focus::placeholder {
  color: #94a3b8;
}
```

## 9. 性能考虑

- 伪元素不会增加DOM节点，性能开销小
- 过度使用复杂的结构伪类（如 `:nth-child` 嵌套）可能会影响性能
- 伪元素的 `content` 属性内容会被屏幕阅读器忽略，需要注意可访问性

## 10. 浏览器兼容性

- 大多数现代浏览器都支持CSS3伪类和伪元素
- 对于旧版IE浏览器，可能需要使用单冒号语法（如 `:before` 而不是 `::before`）
- 某些高级伪类（如 `:nth-child`）在旧浏览器中可能不被支持
