# Lesson 06: 调试利器 (DevTools) - 解剖网页的手术刀

在第一课我们开启了“显微镜”，这一课我们要学习如何用它进行“手术”。

## 1. 解决“样式冲突”：Computed 面板
当你的 CSS 规则很多时，往往会出现“我设置了颜色，为什么它还是黑的？”的情况。
- **操作**：选中元素 -> 点击 Styles 旁边的 **Computed**。
- **作用**：这里展示的是浏览器**最终决定执行**的样式。你可以看到到底是谁（哪个文件、哪一行、哪个选择器）覆盖了你的样式。

## 2. 状态模拟：Force State
如果你想调试鼠标悬停 (`:hover`) 或点击 (`:active`) 时的样式，但手一动悬停就消失了怎么办？
- **操作**：选中元素 -> 在 Styles 面板点击右上角的 **:hov**。
- **作用**：你可以强制勾选 `:hover` 状态，让元素一直处于悬停模式，方便你调试阴影或颜色变化。

## 3. 布局调试：Flexbox & Grid 徽章
现代浏览器会给布局复杂的盒子打上“小徽章”。
- **操作**：在 Elements 面板，看到标签旁边有 `flex` 或 `grid` 单词时，点击它。
- **作用**：浏览器会在页面上画出辅助线，让你看清每一个格子是怎么排列的，这在后续学习高级布局时非常有帮助。

## 4. 实时测量：Box Model 图解
面板最下方的那个四色图：
- **蓝色**：Content (内容)
- **绿色**：Padding (内边距)
- **黄色**：Border (边框)
- **橙色**：Margin (外边距)
- **秘籍**：你可以直接在这个图上双击数字并修改，页面会实时变化。
