# Lesson 04: 颜色与文本 - 让文字会说话

网页的核心依然是信息传递。在这一课，我们将学习如何控制文字的外观，让它不仅清晰易读，而且具备设计感。

## 1. 颜色的表达方式
在 CSS 中，你可以通过多种方式给文字上色：
- **关键字**：`red`, `blue`, `hotpink`（简单但不精准）。
- **十六进制 (Hex)**：`#ff0000` (红), `#0000ff` (蓝), `#ffffff` (白)。最常用。
- **RGB/RGBA**：`rgb(255, 0, 0)`。`a` 代表透明度（0~1），例如 `rgba(0, 0, 0, 0.5)` 是半透明黑色。

## 2. 文本基本属性
- **font-size**：字体大小。目前先用 `px`（比如 `16px`），后面我们会学更高级的单位。
- **font-weight**：字体粗细。常用值有 `normal`, `bold` 或者数字 `400` (正常), `700` (加粗)。
- **font-family**：字体家族。你可以指定多个字体作为备选，浏览器会按顺序查找。
  `font-family: "Microsoft YaHei", sans-serif;`
- **line-height**：行高。决定了行与行之间的距离。推荐使用数字倍数，比如 `1.5`。

## 3. 文本排版属性
- **text-align**：对齐方式。`left` (左), `center` (中), `right` (右), `justify` (两端对齐)。
- **text-decoration**：文本装饰。常用 `none`（去掉超链接下划线）或 `underline`（加下划线）。
- **text-indent**：首行缩进。比如 `2em` 代表缩进两个字。

## 4. 属性的继承 (Inheritance)
这是一个非常重要的概念：如果你给 `<body>` 或者一个 `<div>` 设置了 `color` 或 `font-size`，它里面的所有子元素**默认都会继承**这些样式，除非你单独给子元素设置了新样式。

**任务**：打造一篇精美的排版文章。

1. 打开 `exercises/04_Text/index.html`。

    - 去掉底部链接的下划线，并让它在鼠标悬停时（虽然还没学伪类，可以先试试）变色。
