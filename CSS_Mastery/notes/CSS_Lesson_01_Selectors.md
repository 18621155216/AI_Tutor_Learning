# Lesson 01: 选择器入门 - 告诉浏览器“谁”要变美

在上一课，我们学会了如何让所有的 `<h1>` 变色。但现实中，我们可能只想让**某一个**标题变色，或者让**某一类**文字变色。这时候就需要用到更精准的“定位方式”——**选择器**。

## 1. 标签选择器 (Tag Selector)
最基础的选择器，直接写标签名。它会选中页面上**所有**该标签。
- **语法**：`h1 { ... }`, `p { ... }`, `div { ... }`
- **特点**：全场通杀，范围最广。

## 2. 类选择器 (Class Selector) - 最常用！
如果你想给不同的元素穿上“同一套制服”，就用类选择器。
- **第一步 (HTML)**：在标签里加一个 `class` 属性。
  ```html
  <p class="highlight">我是重点内容</p>
  ```
- **第二步 (CSS)**：在名字前面加一个 **点 `.`**。
  ```css
  .highlight {
    color: orange;
    font-weight: bold;
  }
  ```
- **特点**：非常灵活，一个标签可以有多个类（用空格隔开），一个类也可以给多个标签用。

## 3. ID 选择器 (ID Selector) - 唯一标识
如果你想给某个元素一个“身份证号”，全页面唯一，就用 ID 选择器。
- **第一步 (HTML)**：在标签里加一个 `id` 属性。
  ```html
  <h1 id="main-logo">我的网站 logo</h1>
  ```
- **第二步 (CSS)**：在名字前面加一个 **井号 `#`**。
  ```css
  #main-logo {
    width: 200px;
  }
  ```
- **特点**：**唯一性**。一个 HTML 页面里，同一个 ID 只能出现一次。
