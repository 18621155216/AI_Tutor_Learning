任务 A：特异性计算挑战

不修改 HTML，只修改 CSS，通过调整选择器特异性实现目标颜色

目标1：需要变成绿色
目标2：需要变成蓝色
目标3：需要变成橙色
提示：CSS 中已经写好了各种选择器。你需要：
  1. 为每个目标元素计算当前生效的选择器特异性
  2. 通过添加/修改选择器，让正确的颜色生效
  3. 不要使用 !important，只调整特异性

任务 C：继承控制

使用 inherit、initial、unset 关键字控制样式继承

父元素（有自定义样式）
child-inherit：继承父元素的颜色
child-initial：使用属性的初始值
child-unset：如果是继承属性就继承，否则用初始值
child-normal：普通子元素（对比用）
要求：在 CSS 中为每个 .child 类添加相应样式：
  • .child-inherit：继承父元素的颜色
  • .child-initial：使用字体大小的初始值
  • .child-unset：如果是继承属性就继承，否则用初始值
