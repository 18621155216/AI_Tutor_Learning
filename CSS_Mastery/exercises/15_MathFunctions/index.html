任务 B：clamp() 函数

使用 clamp() 函数创建响应式值

字体大小：clamp(1rem, 2vw, 2rem)
宽度：clamp(200px, 50%, 500px)
内边距：clamp(10px, 5vw, 30px)
外边距：clamp(10px, 2vw, 20px)
要求：
  • 使用 clamp() 函数创建响应式值
  • 理解 clamp(min, preferred, max) 的工作原理
  • 应用于字体大小、宽度、间距等属性

任务 C：min() 和 max() 函数

使用 min() 和 max() 函数选择最小值和最大值

最小宽度：min(50%, 500px)
最大宽度：max(300px, 30%)
最小字体：min(24px, 5vw)
最大字体：max(16px, 2vw)
要求：
  • 使用 min() 函数选择最小值
  • 使用 max() 函数选择最大值
  • 应用于不同的 CSS 属性

任务 D：数学函数组合使用

组合使用多种数学函数

复杂组合：clamp(1rem, calc(1rem + 2vw), 2rem)
网格布局：grid-template-columns: repeat(auto-fit, minmax(min(250px, 100%), 1fr))
响应式间距：padding: clamp(10px, 2vw, 20px)
要求：
  • 组合使用 calc()、clamp()、min()、max() 函数
  • 创建复杂的响应式布局
  • 优化不同屏幕尺寸下的显示效果
