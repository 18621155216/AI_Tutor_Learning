# JavaScript 全栈进阶之路 (Logic & Engineering Mastery)

## 🟢 第一阶段：必须掌握（初级 → 中级就业必备）
> **目标**：掌握 JS 基本语法与逻辑控制，能够处理日常交互数据。

- [x] **01. 变量与数据类型**：彻底搞懂 `let`, `const` 与 `var`；基本类型 vs 引用类型（堆栈内存概念）。
- [ ] **02. 运算符与表达式**：算术、逻辑（短路运算 `&&` `||`）、三元运算符、解构赋值（Destructuring）。
- [ ] **03. 流程控制**：`if-else`, `switch`, `for`/`while` 循环，以及高性能的 `Array.forEach/map/filter`。
- [ ] **04. 函数基础**：函数声明、匿名函数、**箭头函数 (Arrow Functions)** 及参数默认值。
- [ ] **05. 对象与数组操作**：常用 API（push, slice, splice, reduce）、对象属性访问、JSON 数据处理。
- [ ] **06. DOM 基础操作**：选择器（querySelector）、事件监听（addEventListener）、动态修改样式与内容。

## 🟡 第二阶段：应该掌握（中级提升 & 面试高频）
> **目标**：理解 JS 运行机制，具备处理异步数据和复杂逻辑的能力。

- [ ] **07. 作用域与闭包 (Closure)**：全局/局部作用域、静态作用域、闭包的实战场景（私有变量）。
- [ ] **08. 原型与原型链**：`__proto__`, `prototype`，理解 JS 的继承本质（面试重灾区）。
- [ ] **09. 异步编程 A：Promise**：状态机原理、`.then().catch()`、`Promise.all()` 并发处理。
- [ ] **10. 异步编程 B：Async/Await**：如何用同步的写法写异步逻辑（处理后端 API 调用的核心）。
- [ ] **11. 事件流机制**：冒泡 (Bubbling)、捕获 (Capturing)、事件委托 (Delegation)。
- [ ] **12. ES6 模块化 (ES Modules)**：`import` 与 `export`，理解模块化开发思维。

## 🔵 第三阶段：加分项 / 现代特性（2024–2026 大厂标配）
> **目标**：对齐现代前端框架（Vue/React）底层，具备编写插件和工具库的能力。

- [ ] **13. 类与面向对象 (Class)**：构造函数、继承 (extends)、静态方法、私有属性 (#private)。
- [ ] **14. 代理与反射 (Proxy & Reflect)**：Vue 3 响应式的底层原理（如何监听对象变化）。
- [ ] **15. 深浅拷贝与内存管理**：递归实现深拷贝、垃圾回收机制 (GC)、内存泄漏排查。
- [ ] **16. 高阶函数与函数式编程**：纯函数、柯里化、防抖 (Debounce) 与节流 (Throttle) 的手写实现。
- [ ] **17. 现代网络请求**：Fetch API、AbortController (取消请求)、处理二进制数据 (Blob/ArrayBuffer)。
- [ ] **18. 错误处理与调试**：`try-catch-finally` 进阶、自定义 Error、SourceMap 调试技巧。

## ⚪ 第四阶段：了解概念 / 选修方向
> **目标**：扩展技术广度，应对特殊应用场景。

- [ ] **19. 正则表达式 (Regex)**：常用匹配模式、断言、性能陷阱。
- [ ] **20. Web Components**：原生组件化方案，了解 HTML 模板、Shadow DOM。
- [ ] **21. 设计模式**：观察者模式、发布订阅模式、单例模式在 JS 中的应用。
- [ ] **22. 性能监控**：Performance API、Long Tasks 检测、首屏加载时间计算。
