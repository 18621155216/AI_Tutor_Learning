# HTML 工业级全栈开发知识地图 (Complete Guide)

## 1. 核心骨架与资源管理
- **DOCTYPE**: `<!DOCTYPE html>` 触发标准模式。
- **Meta TDK**: 
  - `charset="UTF-8"`
  - `viewport`: 响应式设计基础。
  - `description`/`keywords`: SEO 核心。
- **资源引入**:
  - `link`: CSS (`rel="stylesheet"`), Favicon.
  - `script`: `async` (加载完即执行) / `defer` (DOM解析完按顺序执行).

## 2. 语义化文本与结构
- **块级元素**: `h1-h6`, `p`, `div`, `blockquote`, `pre`, `hr`.
- **行内元素**: `span`, `strong`, `em`, `code`, `mark`, `a`.
- **列表**: `ul/li` (无序), `ol/li` (有序), `dl/dt/dd` (描述列表).
- **布局标签**: `header`, `nav`, `main`, `footer`, `section`, `article`, `aside`.
- **实体字符**: `&lt;`, `&gt;`, `&amp;`, `&nbsp;`.

## 3. 超链接与多媒体
- **超链接**: `<a>`
  - `href`: 相对/绝对路径, 锚点(`#`), 协议(`mailto:`, `tel:`).
  - `target="_blank"` 必须配合 `rel="noopener"`.
- **图像**: `<img>`
  - `alt`: 必填，辅助功能。
  - `loading="lazy"`: 原生懒加载。
  - `srcset`/`sizes`: 响应式适配。
  - `<picture>`: 格式回退。
- **嵌入**: `<video>`, `<audio>`, `<iframe>` (注意 `sandbox`).

## 4. 工业级表单交互
- **表单容器**: `<form>` (action, method).
- **控件绑定**: `<label for="id">` 提升可用性。
- **输入类型**: `text`, `password`, `email`, `tel`, `checkbox`, `radio`, `file`.
- **增强组件**: `select/option`, `textarea`, `datalist`.
- **组织结构**: `fieldset`, `legend`.

## 5. 高级特性与性能
- **表格**: `table`, `thead`, `tbody`, `tfoot`, `tr`, `th`, `td` (使用 `scope` 增强 A11y).
- **SVG**: 矢量图形，推荐内联使用。
- **交互组件**: `<details>/<summary>`, `<dialog>`.
- **A11y (可访问性)**: `aria-*` 属性，键盘焦点管理。
- **客户端存储**: `LocalStorage`, `SessionStorage`.
- **性能提示**: `rel="preload"`, `rel="prefetch"`.
