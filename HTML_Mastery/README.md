# HTML Mastery: Industrial-Grade Learning Journey

本项目记录了我从零基础开始，向 **全栈工程师** 迈进的 HTML 学习历程。

## 🚀 学习亮点
- **语义化 (Semantics)**: 深入理解 `main`, `article`, `section` 等标签的正确用法。
- **可访问性 (A11y)**: 学习 `aria-*` 属性，构建包容性的网页。
- **性能优化 (Performance)**: 掌握 `srcset`, `picture` 以及资源预加载 (`preload`/`prefetch`)。
- **现代交互**: 探索原生 `<dialog>`, `<details>` 和 SVG 矢量图形。

## 📂 目录结构
- `/exercises`: 包含 15+ 个实战练习 (Labs)。
- `/notes`: 每一课的详细技术笔记和架构师建议。
- `progress.json`: 自动化的学习进度跟踪。
- `plan.md`: 完整的 HTML 工业级精通之路路线图。

## 🎯 最终目标
成为一名具备架构思维的全栈工程师。
